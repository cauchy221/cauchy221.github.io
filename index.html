2 Gradient

2.1 What is gradient

梯度gradient: 一个向量，表示某一函数在该点的方向导数沿着该方向取得最大值，即函数在该点沿着该方向变化最快，变化率最大（为该梯度的模）

2.2 Computing Gradient

2.2.1 Numerical gradient

就是自己取一个极小的dx，利用公式 ( f(x+dx)-f(x) )/dx计算

2.2.2 Analytic gradient

使用微积分中的梯度计算公式

2.2.3 Gradient check

  • 实际训练中，我们使用第二种方法计算，但是可以用第一种方法check (debugging tool);
  • Pytorch提供一个类似的方法: grad check

3 Gradient Descent

3.1 Hyperparameters

  • Weight initialization method: 虽然通常认为随机选取初始的W就行，但实际上发现，初始W的选取对训练有不小的影响；
  • Number of steps
  • Learning rate: how big a step do we want to take

3.2 Vanilla Stochastic Gradient Descent (SGD)

在N很大的时候可以使用这种方法简化运算。相当于用一个minibatch-loss上的梯度去近似original-loss

3.2.1 Main idea

3.2.2 Problems with SGD

  • step size (learning rate) 的设置：难以权衡，较大可能会overshooting，较小训练较慢;
  • 有 local minimum or saddle point（后者对于 high dimension 更常见）;
  • Our gradients come from minibatches so they can be noisy

3.3 Smarter SGD: SGD+Momentum（动量）

3.3.1 Main idea

  • Momentum会保留之前步子的趋势这里的 v 可以看成是小球在曲线上摆动的速度，当曲线越陡峭时，下滑速度越大，vice versa;
  • 后面两种方法其实是完全一样的；
  • 这种方法可以解决上面提到的普通SGD不能解决的问题

3.4 SGD+Nesterov Momentum（牛顿动量）

3.4.1 Main idea

  • 它先走到你下一步将要到的那个点，然后把那个 “未来的点” 的梯度计算出来（取代当前点的地位），直接更新动量和x；
  • 优点：学习率合适，如果下一步趋势不变，可以看做等价替换（不会产生放大效应，不会double位移）；如果下一步有加速、减速或者掉头的趋势，又能提前实现（因为能提前找到下一步的点，提前往正确的方向拉回来）;

Lecture 3: Linear Classification

1 Linear Classifier

  • Algebraic Viewpoint: f(x, W) = Wx + b
  • Visual Viewpoint: linear classifier has one “template” per category
  • Geometric Viewpoint: hyperplanes carving up a high-dimensional space

2 Loss Function

It tells how good our current classifier is (low = good, high = bad)

2.1 Multiclass SVM Loss

2.1.1 Some questions

  • Once that an example is correctly classified, than changing the predicted scores of that examples just a little bit don’t really effect the loss
  • min possible loss: 0; max possible loss: infinite
  • The classifier was randomly initialized, what loss would we expect: C - 1, C is the number of categories (Li = C - 1, L = N * Li / N = C - 1)
  • What would happen if the sum were over all classes: add 1
  • What if the loss used a mean instead of a sum: losses multiplied a factor 1/(c-1)

2.1.2 Regularization

  • Express preferences in among models beyond ”minimize training error”
  • Avoid overfitting: Prefer simple models that generalize better
  • Improve optimization by adding curvature

2.2 Cross-Entropy Loss (Multinomial Logistic Regression)

2.2.1 Functions to measure difference between probability distributions

  • Kullback-Leibler Divergence
  • Cross Entropy

2.2.2 Some questions

  • min loss = 0 (can never achieve); max loss = infinity
  • If W is all random values: L = -log(C)

Lecture 2: Image Classification

1 Problem: Semantic Gap

电脑看到的是像素值

补充：对于计算机来说，图像是一个由数字组成的巨大的3维数组。在这个例子中，猫的图像大小是宽248像素，高400像素，有3个颜色通道，分别是红、绿和蓝（简称RGB）。如此，该图像就包含了248X400X3=297600个数字，每个数字都是在范围0-255之间的整型，其中0表示全黑，255表示全白。我们的任务就是把这些上百万的数字变成一个简单的标签，比如“猫”。

2 Challenges

  • Viewpoint Variation 看猫咪的角度不同，机器可能难以识别，但人类知道是猫咪
  • Intraclass Variation 一类物体的个体之间的外形差异很大
  • Fine-Grained Categories 都是猫但品种不同
  • Background Clutter 背景干扰
  • Illumination Changes 光线干扰
  • Deformation 形变
  • Occlusion 遮挡

3 Support Other Tasks

  • Object Detection
  • Image Captioning
  • Playing Go

4 Image Classification Datasets

  • MNIST
  • CIFAR10
  • CIFAR100（本课程通常使用这两个数据集）
  • ImageNet
  • MIT Places
  • Omniglot (20 images per category)

5 Some Classifier

5.1 K-Nearest Neighbor

  • How fast is training: O(1)

  • How fast is testing: O(N) (In fact we can afford slow training, but we need fast testing)

  • Use different Distance Metric to calculate distances (L1 dist. or L2 dist.)

  • Change the value of K in K-Nearest Neighbors (These two are called Hyperparameters)

  • Split data into TRAIN, VALIDATION and TEST sets

  • Better idea: Cross-Validation, split data into folds, try each fold as validation and average the results (commonly used for small datasets, but not used too frequently in DL)
