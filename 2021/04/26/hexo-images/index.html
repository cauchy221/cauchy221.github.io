步骤一 插件安装与设置

安装

首先安装一个插件hexo-asset-image
在hexo目录下git bash安装：npm install https://github.com/CodeFalling/hexo-asset-image --save

步骤二 config设置

打开hexo目录下的_config.yml文件，修改下述内容
post_asset_folder: true
将其改为true后，每次新建一个post，会自动生成一个同名文件夹。只需要将需要插入的图片放进该文件夹内就行

步骤三 图片引用

根据hexo官方文档，.md中的图片引用采用了新的格式，即使用下面的标签插件，而不是md语法
{% asset_img example.jpg This is an example image %}
其中example.jpg为图片的名称，后面为对图片的描述（可以省略）
