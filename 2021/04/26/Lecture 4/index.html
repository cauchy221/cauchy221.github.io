2 Gradient

2.1 What is gradient

梯度gradient: 一个向量，表示某一函数在该点的方向导数沿着该方向取得最大值，即函数在该点沿着该方向变化最快，变化率最大（为该梯度的模）

2.2 Computing Gradient

2.2.1 Numerical gradient

就是自己取一个极小的dx，利用公式 ( f(x+dx)-f(x) )/dx计算

2.2.2 Analytic gradient

使用微积分中的梯度计算公式

2.2.3 Gradient check

  • 实际训练中，我们使用第二种方法计算，但是可以用第一种方法check (debugging tool);
  • Pytorch提供一个类似的方法: grad check

3 Gradient Descent

3.1 Hyperparameters

  • Weight initialization method: 虽然通常认为随机选取初始的W就行，但实际上发现，初始W的选取对训练有不小的影响；
  • Number of steps
  • Learning rate: how big a step do we want to take

3.2 Vanilla Stochastic Gradient Descent (SGD)

在N很大的时候可以使用这种方法简化运算。相当于用一个minibatch-loss上的梯度去近似original-loss

3.2.1 Main idea

3.2.2 Problems with SGD

  • step size (learning rate) 的设置：难以权衡，较大可能会overshooting，较小训练较慢;
  • 有 local minimum or saddle point（后者对于 high dimension 更常见）;
  • Our gradients come from minibatches so they can be noisy

3.3 Smarter SGD: SGD+Momentum（动量）

3.3.1 Main idea

  • Momentum会保留之前步子的趋势这里的 v 可以看成是小球在曲线上摆动的速度，当曲线越陡峭时，下滑速度越大，vice versa;
  • 后面两种方法其实是完全一样的；
  • 这种方法可以解决上面提到的普通SGD不能解决的问题

3.4 SGD+Nesterov Momentum（牛顿动量）

3.4.1 Main idea

  • 它先走到你下一步将要到的那个点，然后把那个 “未来的点” 的梯度计算出来（取代当前点的地位），直接更新动量和x；
  • 优点：学习率合适，如果下一步趋势不变，可以看做等价替换（不会产生放大效应，不会double位移）；如果下一步有加速、减速或者掉头的趋势，又能提前实现（因为能提前找到下一步的点，提前往正确的方向拉回来）;
