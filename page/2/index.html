2 Python

2.1 Basic Data Types

  • int, float, boolean, string etc.

2.2 Containers

2.2.1 List

https://docs.python.org/3.5/tutorial/datastructures.html#more-on-lists

  • The python equivalent of an array, but is resizable and can contain elements of different types
  • Slicing
  • Loops
  • If you want access to the index of each element within the body of a loop, use the built-in enumerate function
  • List Comprehension

2.2.2 Dictionaries

https://docs.python.org/3.5/library/stdtypes.html#dict

  • A dictionary stores (key, value) pairs
  • Loops
  • If you want access to keys and their corresponding values, use the items method
  • Dictionary comprehensions

2.2.3 Sets

https://docs.python.org/3.5/library/stdtypes.html#set

  • A set is an unordered collection of distinct elements
  • Loops (sets are unordered)
  • Set comprehension

2.2 4 Tuples

  • A tuple is an (immutable) ordered list of values. A tuple is in many ways similar to a list; one of the most important differences is that tuples can be used as keys in dictionaries and as elements of sets, while lists cannot. Tuple is also unchangeable.

2.3 Functions

  • Python functions are defined using the def keyword

2.4 Classes

3 Numpy

http://docs.scipy.org/doc/numpy/reference/

It provides a high-performance multidimensional array object, and tools for working with these arrays

3.1 Arrays

http://docs.scipy.org/doc/numpy/user/basics.creation.html#arrays-creation

  • Rank: The number of dimensions
  • Shape: A tuple of integers giving the size of the array along each dimension

3.2 Array indexing

http://docs.scipy.org/doc/numpy/reference/arrays.indexing.html

  • Slicing
  • Integer array indexing
  • Boolean array indexing

3.3 Datatypes

  • Every numpy array is a grid of elements of the same type
  • Can be constructed explicitly or implicitly

3.4 Array math

  • * 对应元素相乘
  • dot 矩阵乘法
  • T 矩阵转置 (the transpose of a rank 1 array does nothing)

3.5 Broadcasting

http://docs.scipy.org/doc/numpy/user/basics.broadcasting.html

http://scipy.github.io/old-wiki/pages/EricsBroadcastingDoc

  • without broadcasting
  • using broadcasting allows us to perform this computation without actually creating multiple copies of v

4 SciPy

http://docs.scipy.org/doc/scipy/reference/index.html

SciPy provides a large number of functions that operate on numpy arrays and are useful for different types of scientific and engineering applications

4.1 Image operations

  • read images from disk into numpy arrays
  • write numpy arrays to disk as images
  • resize images

4.2 MATLAB files

  • scipy.io.loadmat
  • scipy.io.savemat

4.3 Distance between points

  • The function scipy.spatial.distance.pdist computes the distance between all pairs of points in a given set
  • A similar function scipy.spatial.distance.cdist computes the distance between all pairs across two sets of points

5 Matplotlib

http://matplotlib.org/

A plotting library

5.1 Plotting

  • Function plot allows you to plot 2D data

5.2 Subplots

  • Can plot different things in the same figure using the subplot function

5.3 Images

  • You can use the imshow function to show images
